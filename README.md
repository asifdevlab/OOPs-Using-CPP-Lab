# OOPs Using C++ Lab Programs

This repository contains all laboratory programs for the course **Object Oriented Programming using C++**.

## Experiments Included

1. C++ Program Structure and I/O
2. Control Structures
3. Functions and Function Overloading
4. Arrays and Strings
5. Classes and Objects
6. Constructors and Destructors
7. Encapsulation and Data Hiding
8. Inheritance
9. Polymorphism and Virtual Functions
10. Abstract Classes
11. Exception Handling
12. File Handling

Each experiment folder contains the required C++ programs according to the lab manual.
